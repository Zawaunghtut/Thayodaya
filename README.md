# သရောဒယ အဆင့်မြင့်တွက်စက် (Swarodaya Advanced Calculator) 🔮

မြန်မာ့ရိုးရာ သရောဒယပညာရပ်ကို အခြေခံ၍ လူအမည်၊ ဘုရားဘွဲ့တော်နှင့် လှူဖွယ်ပစ္စည်းတို့၏ ဓာတ်သဟဇာတဖြစ်မှုကို အဆင့်မြင့်စွာ တွက်ချက်ပေးနိုင်သော Web Application ဖြစ်ပါသည်။

## 📌 နိဒါန်း (Introduction)
ဤ Project သည် ရှေးဟောင်းမြန်မာ့ကျမ်းဂန်လာ သရောဒယနည်းစနစ်များကို ခေတ်မီနည်းပညာဖြင့် ပေါင်းစပ်ဖန်တီးထားခြင်း ဖြစ်ပါသည်။ အလှူအတန်းပြုလုပ်ရာတွင် သော်လည်းကောင်း၊ အမှုကိစ္စများ ဆောင်ရွက်ရာတွင် သော်လည်းကောင်း မိမိနှင့် သင့်တော်သော ဘုရားဘွဲ့တော်နှင့် လှူဖွယ်များကို အလွယ်တကူ ရွေးချယ်တွက်ချက်နိုင်ရန် ရည်ရွယ်ပါသည်။

## ✨ အဓိက လုပ်ဆောင်ချက်များ (Key Features)
* Precise Calculation: သရောဒယ နည်းစနစ်အတိုင်း တိကျသော တွက်ချက်မှု။
* User-Friendly Interface: အမည်းရောင်နှင့် ရွှေရောင်၊ အနီရင့်ရောင်တို့ကို အချိုးကျ အသုံးပြုထားသော Premium Design။
* Fast Performance: တွက်ချက်မှုများ မြန်ဆန်ပြီး ပေါ့ပါးခြင်း။
* Fully Responsive: ဖုန်း၊ တက်ဘလက်နှင့် ကွန်ပျူတာ အားလုံးတွင် အဆင်ပြေစွာ အသုံးပြုနိုင်ခြင်း။

## 🛠 အသုံးပြုထားသော နည်းပညာများ (Tech Stack)
* HTML5
* CSS3 (Custom UI Styling)
* JavaScript (Calculation Logic)
